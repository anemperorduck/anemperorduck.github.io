使用hexo搭建自己的主题博客

Hexo 是一个快速、简洁且高效的静态博客框架，基于Node.js构建，支持Markdown语法，非常适合用来搭建个人博客。本教程将带你从零开始，一步步搭建并部署你的第一个Hexo博客。

Hexo 基于 Node.js 运行，需要安装 Node.js（建议 LTS 版本）。

使用 GitHub 获取免费域名（GitHub Pages 部署）

GitHub 提供免费静态网站托管服务 GitHub Pages，非常适合部署 Hexo 博客。

1. 创建 GitHub 仓库

  • 登录 GitHub
  • 创建新仓库，命名为 你的用户名.github.io（例如：zhangsan.github.io）
  • 仓库名必须严格符合 用户名.github.io 格式，才能直接通过该地址访问
